# Contributing Guide

Thank you for your interest in contributing to the project!

This document describes how to contribute code, report bugs, create pull requests, and how maintainers will review.

---

## 🧪 1. General Requirements

- Node.js >= 18
- pnpm / npm / yarn
- MongoDB / PostgreSQL (depending on the project)
- Code review with ESLint before commit

---

## 🐛 2. Bug Report

When creating an issue, please include:

- Environment (OS, Browser, Node version…)
- Steps to reproduce the bug
- Expected and actual results
- Log / Screenshot (if available)

Create an issue at:
`https://github.com/<your-repo>/issues`

---

## ✨ 3. Feature Request

Please describe:

- Feature you want to add, reason and impact
- API changes
- Mockup image (if UI)
